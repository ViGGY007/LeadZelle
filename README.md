# LeadZelle Website

## Setup
No build step required. Pure HTML/CSS/JS.

## Local Development
Open any .html file directly in a browser.
Or use Live Server VS Code extension for hot reload.

## Deployment
1. Push all files to GitHub repository
2. Connect repository to Vercel
3. Vercel auto-deploys on every push
4. vercel.json handles clean URL routing

## File Structure
- index.html — Homepage
- how-it-works.html — How It Works page
- services.html — Services page
- industries.html — Industries page
- demo.html — Demo page with Typebot embed
- pricing.html — Full pricing page
- contact.html — Contact page
- styles.css — All shared styles
- animations.js — All shared GSAP animations
- vercel.json — Vercel routing config
